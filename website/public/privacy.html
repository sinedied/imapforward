Privacy Policy

Last updated: April 9, 2026

1. Overview

imapforward is a self-hosted, open-source email forwarding tool. It runs entirely on your own infrastructure. We do not operate any servers that process your data.

2. Data Collection

The Software does not collect, store, transmit, or share any personal data with the authors or any third party. All email data is processed locally on the machine where you run the Software.

3. Credentials

Email account credentials and OAuth2 tokens are stored exclusively in your local configuration file. They are never sent anywhere other than to the email providers you configure (IMAP/SMTP servers or the Gmail API).

4. Gmail API Usage

When using the Gmail API forwarding method, the Software accesses your Gmail account using OAuth2 credentials that you create and control. The Software:

  • Only requests the minimum required permissions (gmail.insert and gmail.labels)
  • Only imports messages into your mailbox and manages labels
  • Does not read, delete, or modify existing emails in your account
  • Does not share your data with any third party

Google's own privacy practices are described in the Google Privacy Policy.

5. Website

The imapforward website (sinedied.github.io/imapforward) is a static site hosted on GitHub Pages. It does not use cookies, analytics, or tracking of any kind. The configuration generator runs entirely in your browser — no data is sent to any server.

6. Changes

This policy may be updated at any time. Changes will be reflected on this page with an updated date.

7. Contact

For questions regarding this policy, open an issue on the GitHub repository.
